# @univerjs/docs-hyper-link

## Introduction

`@univerjs/docs-hyper-link` provides the base link function of Univer Docs.

## Usage

### Installation

```shell
# Use npm
npm install @univerjs/docs-hyper-link

# Use pnpm
pnpm add @univerjs/docs-hyper-link
```
