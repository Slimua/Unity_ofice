## Introduction

`@univerjs/sheets-sort` is a sort plugin for Univer Sheet.

## Usage

You should use this plugin with the `@univerjs/sheets-sort-ui` package.

### Installation

```shell
# Using npm
npm install @univerjs/sheets-sort

# Using pnpm
pnpm add @univerjs/sheets-sort
```
