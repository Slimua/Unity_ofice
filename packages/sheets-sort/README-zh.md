## 简介

`@univerjs/sheets-sort` 是 Univer Sheet 的排序插件。

## 使用指南

你应该将此插件与 `@univerjs/sheets-sort-ui` 包一起使用。

### 安装

```shell
# 使用 npm
npm install @univerjs/sheets-sort

# 使用 pnpm
pnpm add @univerjs/sheets-sort
```
