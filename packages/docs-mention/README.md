# @univerjs/docs-mention

## Introduction

`@univerjs/docs-mention` package provides the Mention for Docs.

## Usage

### Installation

```shell
# Using npm
npm install @univerjs/docs-mention

# Using pnpm
pnpm add @univerjs/docs-mention
```
